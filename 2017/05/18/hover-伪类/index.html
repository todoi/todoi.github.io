hover 伪类

发表于 2017-05-18 分类于 CSS

简介

当鼠标浮动到div标签上的时候，div标签才会显示出background:red的样式。为了确保生效，:hover规则需要放在:link和:visited规则之后，但是在:active规则之前。

注意: 在触摸屏上 :hover 有问题，基本不可用。不同的浏览器上:hover 伪类表现不同。
